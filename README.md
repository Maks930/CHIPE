# CHIP-8 Emulator (C++)

Реализация классической виртуальной машины CHIP-8 на языке **C++**.

## 🛠 Технологический стек
* **Язык:** C++20
* **Графика:** [OpenGL](https://www.opengl.org/) (через GLEW и GLFW)
* **Интерфейс отладки:** [Dear ImGui](https://github.com/ocornut/imgui)
* **Звук:** [OpenAL](https://www.openal.org/)

## 🚀 Особенности
- **Дебаггер:** С помощью ImGui реализованы окна мониторинга:
    - Состояние всех 16 регистров.
    - Текущее значение `Program Counter` и `Index Register`.
    - Визуализация таймеров.
- **Настройка частоты:** Возможность изменять скорость эмуляции (CPU Clock) прямо во время игры.
- **Звуковое сопровождение:** Генерация звукового сигнала через OpenAL при ненулевом значении Sound Timer.

## ⌨️ Управление (Keymap)
Оригинальная клавиатура CHIP-8 (сетка 4x4) сопоставлена с клавишами ПК для удобства игры:

| CHIP-8 | PC Key | CHIP-8 | PC Key |
| :---: | :---: | :---: | :---: |
| **1** | 1 | **2** | 2 |
| **3** | 3 | **C** | 4 |
| **4** | Q | **5** | W |
| **6** | E | **D** | R |
| **7** | A | **8** | S |
| **9** | D | **E** | F |
| **A** | Z | **0** | X |
| **B** | C | **F** | V |
